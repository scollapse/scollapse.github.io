JavaScript基础知识

1.JS概述

JS的作用：给浏览器指令，负责与浏览器沟通

描述的是语言的语法和基本对象，还包括DOM、BOW操作

2.JS的编写位置

  1. HTML代码行内(不推荐)

  1. script标签中

  1. 单独的script文件，文件的后缀是.js

    加载外部js文件

    < script src=“ ”>

注意事项：

  1. script不能写成单标签

  2. 可以省略type属性

  3. 加载顺序

    作为html文件一部分，js默认遵循html文档的加载顺序，自上而下的加载顺序

    建议放在body的最后一行，等页面元素加载完再产生互动效果

  1. js代码严格区分大小写

3.js支持注释：

  1. // 单行注释

  2. /* */ 多行注释

  3. /** **/ 文档注释

4.浏览器的交互方式

方式一：弹出弹窗显示内容

  • alert函数：接收一个参数

  • alert（‘显示内容’）

    方式二：控制台打印

conso.log(内容)

方式三：将内容输入到界面显示

通过document.write（写入的内容）对象

方式四：接收用户输入的内容

prompt(‘提示信息’)，它会返回一个值，即你输入的数据

5.变量

格式：

var 变量名 = 变量赋值；

变量的命名规则：

  1. 第一个字符必须是一个字母或下划线_或$美元符号
  2. 其他字符可以是其他的
  3. 不能使用关键字

变量的命名规范：

  1. 驼峰命名法
  2. 一条语句结束后加上分号；
  3. 赋值=两边都加上空格（可以用格式化文档）

思考：如何交换两个变量（三种方式）

6.数据类型

如何查看数据或变量的类型：

​ typeof操作符

常见数据类型：

  1. 数值型（number）

    数据的表示方式：

    • 十进制
    • 二进制 0b
    • 十六进制 0x
    • 八进制

    数字的表示范围：

    • 最小值：MIN_VALUE
    • 最大值：MAX_VALUE

    NaN：特殊的数值

  1. 字符串(String)

    用于表示一段文字，获取字符串长度的属性：length

  1. 未定义(Undefined)

    只有一个值：undefined

    声明变量不进行赋值的时候的类型

  2. 布尔(Boolean)：只有两个值

    用于表示真假

  3. 空类型（Null）

    通常当一个对象不再使用，可以赋值为null

    Null和Undefined之间的关系：

    值undefined实际上由null引申出来的，undefined和null相等，会返回true

    如果转成数字，undefined会为NaN，null为0

7.转换数据类型

转换成数字型：

其他类型转成数字型 ：

转换成功：获得对应的数字

转换失败：获取的结果是NaN

方法：

Number（any）函数：将任意类型转换成数字

布尔：true 1 ，false 0

数字字符串：对应的数字

非数字的字符串：NaN

parseInt(String)：将字符串转换成整型

如果第1个字符是数字，开始解析，只取前面的数字部分

如果第1个字符不是数字，不解析，返回NaN

如果遇到浮点型，只转换小数点前的整数部分

parseFloat(String)：将字符串转换层浮点型

与上面的用法相似

转换成字符串类型

  1. 调用toString（）方法

    var num = 123;

    console.log(typeof num)

    var str = num.toString();

  1. String(内容)

    ps：toString只能将数值和布尔类型转换成字符串，不能将Undefined和Null转换

    String可以将所有类型转换成字符串类型

  2. 字符串的拼接

    +：+的两侧只要有一侧为字符串，它会把另一侧转换成字符串进行拼接

转换成布尔类型

Boolean（）函数转换成布尔类型

转换成false的五种特殊性：

“ ” ，0 ，undefined ， null ， NaN

除了上面的五种之外，其余的都是true

8.运算符

  • n + +
    • 先赋值，后自加
  • ++n
    • 先自加，后赋值

赋值预算符： =

扩展运算符：

+= -= *= /=

a+=b ： a = a+b

关系比较运算符：

  • 结果：布尔类型

  • 比较大小和相等关系： > < >= <= != == ===

逻辑运算符：

  • 多个条件放在一起进行运算

  • 结果：布尔类型

  • &&逻辑与 ， ||逻辑或 ，！逻辑非

9. 程序的执行顺序

顺序 —从上而下，顺序执行

选择—根据判断条件，决定执行代码的分支

循环—让特定的代码重复的执行

代码块：多行语句的集合，通过一个{}放在一起

选择：

  • 单选 if
  • 多选 switch
