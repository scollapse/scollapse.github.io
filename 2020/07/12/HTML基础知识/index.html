1.HTML的基本标签

P标签

  • 用于将大段的文本分段

br换行 （独目标签）

  • 用于换行

hr横线标签

  • 显示横线

pre 预留格式

  • 代码怎么打的网页就怎么显示

特殊字

  • del删除字
  • ins插入字
  • b粗体字
  • i斜体字
  • sup上角字
  • sub下角字

font字体标签

2.html的实体符号

  • 大于小于号与标签的符号重合，实体符号以&开始，以；结尾

  • 小于：&lt；

  • 大于：&gt；

  • 空格：&nbsp；

3.html的表格

  • table是表格
  • tr是行
  • td是列
  • th是标题表格
  • 、 和 元素：通过使用这些元素，使浏览器有能力支持独立于表格表头和表格页脚的表格主体滚动。当包含多个页面的长的表格被打印时，表格的表头和页脚可被打印在包含表格数据的每张页面上。

属性

  • border
  • width：有px和%，%按照当前窗口的百分比计算，动态改变
  • align：对table用表格居中，对tr，td使用元素居中

单元格合并

  • 行合并：将不要的单元格删掉，保留的使用rowspan合并单元格
  • 列合并： 列删谁都行

4.背景颜色和背景图片

  • bgcolor ： 背景颜色
  • background ：背景图

5.图片

  • src：图片的路径
  • width设置宽度，height设置高度，在设置宽度的时候，高度会等比例缩放；在设置高度的时候，宽度不会，因此可能会发生变形
  • title设置鼠标悬停时显示的信息
  • alt表示图片显示不出来时显示的信息

6.超链接

  • herf ：热引用地址
  • 超链接中间可以加img
  • target：可取值blank（新窗口），self（自己），top（顶层），parent（父）

7.表单

  • form格式
  • action：提交信息的链接
  • method：get或post属性
  • input 标签：（name大多数必须，value值如果用户自写就不用写上，否则需要将value标出来）
    • type：
      • text，文本
      • password：隐藏，作为密码提交
      • submit：提交
      • file：选择文件
      • reset：重置
      • mutiple：多选
      • hiden：隐藏
      • radio：单选
      • checkbox：选择box
  • select：选择
  • textarea：文本框（带滑动条）
  • readonly和disable属性，都不能看到，disable数据无法提交
  • maxlength设置输入框的最多输入字符

8.元素的ID

  • ID是每一个元素标签的唯一标识，在后期获取节点时可以使用ID获得
  • ID不能重复
